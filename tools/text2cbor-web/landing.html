How it Works

1

Upload HTML.zip

Package your static site — or point us to your URL — and we extract the meaningful content structure.

2

Auto-Convert

Our Rust engine transforms HTML into compact, typed CBOR — preserving semantics, stripping noise. Takes seconds.

3

Deploy Alongside

Serve the .cbor file next to your site. AI agents fetch it via Accept: application/cbor — zero changes to your HTML.
