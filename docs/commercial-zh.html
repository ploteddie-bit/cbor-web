CBOR-Web v3.0 架构

根目录下一个 index.cbor 文件。7 个顶级键：

键内容必需作用
0 : @type"cbor-web"是标识
1 : @version3是可扩展性

为什么选择 CBOR 而不是 JSON？

JSON 是文本格式、非确定性的，没有字节字符串和语义标签。CBOR 是由 JSON 的同一作者制定的 IETF 标准（STD 94），用于 WebAuthn、COSE、CWT 和 IoT。确定性编码保证相同内容产生相同字节——可复现的哈希值、可靠的缓存。

生态系统兼容性

CBOR-Web 补充了 robots.txt、sitemap.xml、llms.txt、JSON-LD、OpenAPI。它添加了缺失的层：完整、结构化的原生二进制内容。一次 GET 替代多页面爬取。
